## Overview
This project analyzes customer churn using Power BI to identify churn patterns,
high-risk customer segments, and revenue impact.

## Dataset
Telecom customer dataset containing information on customer demographics,
services, billing, contract types, and churn status.

## Tools Used
- Power BI
- Power Query


## Analysis Performed
- Data cleaning and transformation using Power Query
- Churn rate and retention analysis
- Segmentation by contract type, tenure, services, and billing
- Interactive dashboards using cards, charts, and slicers

## Key Insights
- Higher churn observed in month-to-month contracts
- Tenure strongly influences churn probability
- Certain services and payment methods show higher churn risk
